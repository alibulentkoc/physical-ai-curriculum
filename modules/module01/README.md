## What this module is

Module 1 establishes the mathematical and computational substrate that every later module depends on. It answers one question: **what mathematics does a robot need before it can perceive and act in the physical world?**

Every concept is motivated by the **Greenhouse Harvesting Robot** — representing a fruit's position, relating a camera's view to a gripper's reach, turning joint angles into tool positions — so no equation appears without a physical reason.
