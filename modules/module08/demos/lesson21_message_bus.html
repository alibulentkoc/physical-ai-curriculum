Message Bus — the loop as messages, and where latency comes from

A state message travels from the sensor to the controller, which publishes a command to the actuator, which moves the joint — sensed again on the feedback hop. Each hop takes time; the loop latency is their sum, and it is how old the measurement is by the time the controller acts. Set the hop times and press Send one loop.
