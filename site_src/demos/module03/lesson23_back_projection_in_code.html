Set a pixel and its depth. Back-projection inverts K: xₙ = (u−cₓ)/fₓ, then P_c = Z·(xₙ, yₙ, 1). The round-trip re-projects to the same pixel. Depth = 0 means a sensor miss.
Two checks make back-projection safe: a valid depth (Z > 0), and the re-projection landing back on the input pixel. If the depth sensor returns 0, there's no point to recover.
