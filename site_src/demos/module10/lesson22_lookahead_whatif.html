Each future is a run-ahead of the existing harvester inside the twin — no learning, no statistical model. Because every run shares the same synced start and the same seed, any difference between futures is attributable to the scenario, not to noise. Comparing futures like this is the seed of action selection (Unit 7). Remember: every forecast is only as faithful as the twin (Lesson 6.3).
