Pick an arm. J maps joint-rate space (left) to task space (right). Each splits in two: row space → column space (the useful part), null space → 0 (self-motion), and the left null space (task directions no joint rate can reach).
The pseudoinverse maps column space back to row space, kills the left null space, and adds nothing in the null space — which is why null-space motion is "free" for the task.
