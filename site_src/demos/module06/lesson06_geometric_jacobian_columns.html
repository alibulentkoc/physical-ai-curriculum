Pose the arm. Each Jacobian column is one joint's tip-velocity contribution: column 1 = z₁×(p_tip − p_base), column 2 = z₂×(p_tip − p_elbow). Perpendicular to the link from that joint to the tip.
Build the Jacobian column by column: each revolute joint contributes a velocity z×r, the axis crossed with the vector from joint to tip. ẋ is just θ̇₁·col₁ + θ̇₂·col₂.
