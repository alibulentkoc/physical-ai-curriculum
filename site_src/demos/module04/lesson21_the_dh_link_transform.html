Set the four DH numbers. One transform is built from θ, d, a, α. With d = 0 and α = 0 it collapses to a planar rotation + link translation; push α to 90° and the out-of-plane entries (highlighted) wake up.
θ and d act along the joint axis; a and α describe the link to the next axis. The single four-factor product encodes the whole hop from one joint frame to the next.
