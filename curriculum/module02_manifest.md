# Module 2 — Manifest: Spatial Transformations and SE(3)

## What this module is

Module 1 built the foundations: physical quantities, vectors, coordinate frames, and transformations as *actions on space*. **Module 2 teaches how those transformations are represented and composed in robotics** — the machinery a robot actually uses to move a point from a camera's view into the world and back.

The arc across the curriculum so far:

- **Module 1, Unit 3:** *different observers describe the same point differently.*
- **Module 1, Unit 4:** *transformations change space (a matrix is an action).*
- **Module 2:** *how transformations are represented and composed in robotics* — homogeneous coordinates, rigid-body transforms, SE(2) and SE(3), and transformation chains.

By the end, a student understands homogeneous coordinates, rigid-body transformations (rotation + translation together), rotation matrices, translation vectors, transformation chains, and SE(2)/SE(3) — **without yet performing full robot kinematics** (that is a later module). Module 2 is the representational backbone kinematics will stand on.

## The one new idea that unlocks the module

Module 1 deliberately could not put translation into a matrix (pure 2×2 matrices fix the origin). Module 2's central device is **homogeneous coordinates**: add a coordinate so that translation *also* becomes a matrix multiply. Then rotation + translation combine into a single **rigid-body transformation**, and chains of them compose by multiplication — the structure of **SE(2)** (planar) and **SE(3)** (full 3D rigid motion).

## Educational stance (unchanged from Module 1)

Five-layer flow on every lesson: **Physical Intuition → Visual Understanding → Mathematical Formulation → Computational Implementation → System Integration.** Lead with physical movement and changing viewpoints; do **not** open the module with matrix algebra. The greenhouse-harvesting-robot narrative continues.
