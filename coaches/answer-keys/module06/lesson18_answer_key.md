# Answer Key 5.2
**Q1 B · Q2 B · Q3 B · Q4 B · Q5 B.**
**Q6.** Realizing a tool velocity needs $\dot q=J^{-1}\xi$; a component along the shrinking axis is divided by a vanishing $\sigma_{\min}$, so $\dot q$ explodes — only along the lost direction.
**Q7.** It sits inside the reachable region where a path looks smooth and modest, yet $1/\sigma_{\min}$ ramps up as the path nears it — spiking joint rates without any obvious workspace-edge warning.
**Q8.** $\lVert\dot q\rVert \sim 1/\sigma_{\min}\to\infty$ as $\sigma_{\min}\to 0$.
### Watch for: treating the singularity as one point (it's a hazardous neighborhood); ignoring direction-dependence.
