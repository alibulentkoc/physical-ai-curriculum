# Answer Key 5.3
**Q1 B · Q2 B · Q3 B · Q4 B · Q5 B.**
**Q6.** It makes J block-triangular ($\det J=\det J_{11}\det J_{22}$), so arm ($J_{11}$) and wrist ($J_{22}$) singularities can be tested independently on two 3×3 blocks.
**Q7.** $J_{22}$ (the wrist block) vanishes; an **orientation** direction is lost (two wrist axes do the same rotation).
**Q8.** Elbow = arm fully extended/folded (a position direction along the reach); shoulder = wrist center on the base axis (base rotation stops contributing horizontal wrist-center motion). Both lose **position**, different postures.
### Watch for: assuming all 6-DOF arms decouple; swapping which type loses position vs orientation.
