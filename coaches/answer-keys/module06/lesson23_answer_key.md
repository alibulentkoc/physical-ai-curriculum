# Answer Key 6.3
**Q1 B · Q2 B · Q3 B · Q4 B · Q5 B.**
**Q6.** Row space (joint motions that move the tool); null space (self-motion, tool still); range (reachable tool directions); left-null space (impossible tool directions).
**Q7.** Row space = v₁..v_r; null space = v_{r+1}..v_n; range = u₁..u_r; left-null = u_{r+1}.. (the zero-gain U columns).
**Q8.** They give computable bases for available (range), internal (null space), and impossible (left-null) — the Unit 4 capability buckets, now exact.
### Watch for: confusing null space (joint) with left-null (task); assuming left-null always empty.
