# Module 2 · Lesson 4.3 — Answer Key (instructor only)
**Q1** 4×4 homogeneous matrix. **Q2 (match)** 3×3=rotation; last column=translation; bottom row=[0 0 0 1]. **Q3 (T/F)** True. **Q4 (short)** orientation from the 3×3 block, position from the last column; bottom row [0 0 0 1].
**Challenge (§10) rubric:** bottom row [0 0 1 1] is invalid; it must be [0 0 0 1] for a valid homogeneous rigid transform.
