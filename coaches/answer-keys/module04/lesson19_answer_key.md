# Answer Key — Lesson 5.3: Assigning Frames

**Coaches only.** Formative.

1. **z-axis points** — along the joint's axis of motion.
2. **x-axis points** — along the common normal between consecutive joint axes.
3. **Parallel axes → α = 0** — True.
4. **Perpendicular intersecting axes (short).** a = 0 (axes intersect); α = 90° (perpendicular).

**Challenge rubric.** Full credit: skew lines (neither parallel nor intersecting) have a unique shortest connecting segment — the common normal — which always exists; this is the general case where d, a, θ, α can all be nonzero. Partial: states the common normal exists without the "all four nonzero" conclusion.
