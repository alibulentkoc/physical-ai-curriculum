# Answer Key — Lesson 1.2: Links and Joints

**Coaches only.** Formative.

1. **Revolute joint** — rotates (variable is an angle θ).
2. **DOF equals** — the number of joints (serial arm).
3. **Prismatic slides, variable d** — True.
4. **2R+1P arm (short).** 3 DOF; configuration (θ1, θ2, d3).

**Challenge rubric.** Full credit: ties 6 DOF to the 6 parameters of SE(3) (3 position + 3 orientation); with 4–5 DOF the gripper cannot achieve arbitrary orientation at an arbitrary position (loses orientation freedom). Partial: states "6 needed" without the SE(3) reasoning.
