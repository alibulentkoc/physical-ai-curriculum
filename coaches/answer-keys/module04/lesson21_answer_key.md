# Answer Key — Lesson 6.1: The DH Link Transform

**Coaches only.** Formative.

1. **DH link transform** — Rot_z(θ)·Trans_z(d)·Trans_x(a)·Rot_x(α).
2. **d=0, α=0 reduces to** — the planar transform R_z(θ)·Trans_x(a).
3. **Memorize the four-factor product, not the expanded matrix** — True.
4. **Four motions in order (short).** Rotate about z (θ), translate along z (d), translate along x (a), rotate about x (α).

**Challenge rubric.** Full credit: Trans_z(d) and Rot_z(θ) commute (both about the same z-axis), but Trans_x(a) and Rot_z(θ) do not (rotating then translating along x ≠ translating then rotating). So although the first two factors could be swapped, the overall four-factor order still matters because the x-motions don't commute with the z-rotation. Partial: shows one commutation result without the order conclusion.
