# Answer Key — Lesson 1.3: Configuration vs. Pose

**Coaches only.** Formative.

1. **Configuration is** — the vector of joint variables (joint space).
2. **FK is generally** — many-to-one.
3. **Single-valued forward** — True.
4. **Two configurations for one point (short).** Elbow-up and elbow-down place the gripper at the same position — the many-to-one nature of FK.

**Challenge rubric.** Full credit: connects FK many-to-one to multiple IK solutions and notes the practical benefit (alternate configurations let the arm avoid obstacles while reaching the same fruit). Partial: links to multiple IK solutions without the benefit.
