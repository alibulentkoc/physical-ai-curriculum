# Answer Key — Lesson 5.2: The Four DH Parameters

**Coaches only.** Formative.

1. **Parameters about the joint z-axis** — θ (rotate) and d (slide).
2. **Revolute joint variable** — θ.
3. **Match** — θ → rotate about z; d → slide along z; a → length along x; α → twist about x.
4. **Planar 2-link parameters (short).** d=0 and α=0; a = link length; θ is the variable (revolute).

**Challenge rubric.** Full credit: nonzero d *and* a means the link both steps along the joint axis (offset d) and extends sideways to the next axis (length a) — an L-shaped / stepped link that goes up-and-over. Partial: describes one of the two displacements only.
