# Answer Key — Lesson 3.2: Composing the Chain

**Coaches only.** Formative.

1. **Two-joint FK** — T_0^2 = T_0^1 · T_1^2.
2. **Composition rotation** — θ1 + θ2.
3. **Composition is non-commutative** — True.
4. **Why the product over trig (short).** It scales to any number of joints and to 3D with one rule, while hand-summing angles becomes intractable.

**Challenge rubric.** Full credit: shows associativity (grouping doesn't change the product) but non-commutativity (T_0^1 T_1^2 ≠ T_1^2 T_0^1); physical reading — applying the elbow before the shoulder describes a different arm, so base→tip order is mandatory. Partial: states non-commutativity without the physical interpretation.
