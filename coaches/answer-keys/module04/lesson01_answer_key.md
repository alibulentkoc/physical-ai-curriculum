# Answer Key — Lesson 1.1: The Arm Has to Move

**Coaches only.** Formative.

1. **Forward kinematics maps** — joint angles → gripper pose.
2. **Pose T_0^n gives** — position and orientation of the gripper.
3. **FK supplies Module 3's T(world←arm)** — True.
4. **One-joint positions (short).** θ=0 → (0.5, 0); θ=90° → (0, 0.5).

**Challenge rubric.** Full credit: explains forward is a function (one config → one pose) while inverse can have 0/1/many solutions, with a one-joint example (a point off the radius-L circle is unreachable → 0 solutions; a point on it → generally one orientation).
