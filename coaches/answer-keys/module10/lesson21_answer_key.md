# Answer Key — Lesson 6.1

**MC1:** A — prediction is run-ahead simulation from the current synced state.
**MC2:** A — anchoring to now makes the forecast about the actual current situation.
**MC3:** A — a prediction executes the existing Module 9 system forward in the twin.
**TF4:** True — it runs on a copy, reality untouched, reproducible.
**TF5:** False — prediction introduces no machine learning.

**Short 6 — model answer.** Both monitoring and prediction work because the twin *runs the existing Module 9 system*. Comparing the twin against reality gives monitoring (now); running the twin forward gives prediction (next). The shared property — an executable replica of the real system — is what makes both capabilities available from one model. *Grading:* full credit for identifying the executable-same-system property as the shared basis.
