# Answer Key 2.1 — The Twin's State

**Q1 — B.** The reported world-state: q, tool position, fruit states, health, stage.
**Q2 — B.** Only what the system actually reports.
**Q3 — B.** A copyable, self-contained value.
**Q4 — B.** Because including unreported 'truth' would be fiction, not mirroring.
**Q5 — True.** A faithful twin starts with a faithful state.

**Q6 — model answer.** The twin must be able to hold a state and then *diverge* from the robot (to measure the gap) and later *simulate* it forward independently. An alias of the robot's own objects would change whenever the robot changed — no separate twin state to compare or run, and divergence would read zero by construction. A copyable, self-contained frame gives the twin its own state to mirror, compare, and simulate.
*Grading: credit the divergence/independent-simulation reason.*
