# Answer Key — Lesson 5.4

**MC1:** A — Unit 5 made the twin a live monitor.
**MC2:** A — state (drift) and outcome (surprising behaviour).
**MC3:** A — the predictor asks "what is likely to happen next?"
**TF4:** True — monitoring reused the divergence tools from A and B; no new theory.
**TF5:** False — a monitor is present-tense; forecasting is Unit 6.

**Short 6 — model answer.** A monitor *compares* the twin against reality to answer "what is happening now?" — a present-tense reading of divergence. A predictor *runs the twin forward* to answer "what is likely to happen next?" — a future-tense forecast. Both use the same underlying model; the difference is the question (now vs next) and how the twin is used (compare vs run-forward). *Grading:* full credit for the now/next distinction and same-model point.
