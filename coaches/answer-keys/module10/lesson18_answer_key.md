# Answer Key — Lesson 5.2

**MC1:** A — state divergence rises with drift (stale snapshot).
**MC2:** A — outcome divergence rises with an unmodeled effect (surprising behaviour).
**MC3:** A — quiet state + firing outcome = not staleness; reality behaves unexpectedly → investigate.
**TF4:** True — they are two distinct signals.
**TF5:** False — re-sync clears staleness, not an unmodeled-effect outcome divergence.

**Short 6 — model answer.** Re-sync is right when the *state* channel rises but the outcome matches — the twin's snapshot is merely stale (reality moved between syncs). Investigate/calibrate is right when the *outcome* channel fires (e.g. a fruit predicted-harvested but actually-skipped) — reality carries an effect the twin doesn't model, which a re-sync would not fix. *Grading:* full credit for correctly pairing staleness→re-sync and surprising behaviour→investigate/calibrate.
