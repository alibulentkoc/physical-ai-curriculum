# Answer Key — Lesson 6.3

**MC1:** A — a forecast is only as faithful as the twin (it inherits the sim-to-real gap).
**MC2:** A — calibrated confidence weights a forecast by the twin's measured fidelity.
**MC3:** A — an optimistic forecast (predicts success where reality may fail) needs extra caution.
**TF4:** True — a small, stable monitored gap is evidence to trust a forecast more.
**TF5:** False — calibration shrinks the gap per-effect; a residual always remains.

**Short 6 — model answer.** The monitored Reality ↔ Twin gap is direct evidence of how faithful the twin currently is. A small, stable gap (especially with the relevant effects calibrated) means the twin matches reality, so its forecasts earn more trust. A large or growing gap — or a recent unexplained divergence in the forecast's regime — means the twin is currently wrong there, so the forecast should be doubted. Trust tracks measured fidelity, not the forecast's apparent certainty. *Grading:* full credit for using the monitored gap (and calibration state) as the confidence evidence.
