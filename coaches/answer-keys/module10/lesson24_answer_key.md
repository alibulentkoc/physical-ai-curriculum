**Short 6 — model answer.** The twin is an executable replica of the Module 9 system. Used one way — comparing the twin's mirrored state against reality's live report — it answers "what is happening now?" (monitoring). Used the other way — syncing to now and running the system forward — it answers "what is likely to happen next?" (prediction). One underlying model, two uses, distinguished only by whether you compare against it or run it forward. *Grading:* full credit for the one-model/two-uses explanation tied to compare-vs-run-forward.
