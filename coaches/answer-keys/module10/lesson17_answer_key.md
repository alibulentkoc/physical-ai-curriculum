# Answer Key — Lesson 5.1

**MC1:** A — A monitor answers "what is happening now?"
**MC2:** A — the signal is the divergence between reality and the twin.
**MC3:** A — monitoring reuses `twin.divergence`; no new estimation theory.
**TF4:** True — right after a sync, divergence is ~0 and the monitor is quiet.
**TF5:** False — re-syncing before reading the divergence erases the signal.

**Short 6 — model answer.** Monitoring holds the twin's mirrored state beside reality's live report and reads the difference. It needs no new sensor or estimator because the twin already represents the expected state; the *comparison* between expectation (twin) and observation (reality) is itself the monitor. Divergence within tolerance means "as expected"; divergence beyond tolerance means "something is happening." *Grading:* full credit for framing monitoring as a comparison against an existing reference (the twin) rather than a new measurement device.
