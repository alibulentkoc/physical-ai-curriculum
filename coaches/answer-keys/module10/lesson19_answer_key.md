# Answer Key — Lesson 5.3

**MC1:** A — diagnosis is possible because divergence is localised and directional.
**MC2:** A — which fruit + which direction + which health signal moved.
**MC3:** A — predicted-harvested / actually-skipped is an *optimistic* divergence.
**TF4:** True — health signals characterise *how* a divergence occurred.
**TF5:** False — diagnosis is interpretation of existing signals; no new engine.

**Short 6 — model answer.** A single global error number tells you only *that* something is off, forcing a search for *where* and *what*. The twin's per-fruit, directional, multi-signal divergence names the affected fruit, the direction of the error (optimistic/pessimistic), and which health signal moved — so it points straight at a likely cause (e.g. "obstruction at F3"). Localisation converts detection into diagnosis. *Grading:* full credit for connecting localisation/direction to actionable cause-finding.
