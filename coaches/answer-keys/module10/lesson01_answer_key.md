# Answer Key 1.1 — The Twin Concept

**Q1 — B.** A paired, live, mirroring virtual replica of a specific real system.
**Q2 — B.** Observe, experiment, and reason without touching the real robot.
**Q3 — B.** The Module 9 integrated greenhouse harvester.
**Q4 — B.** None — it reuses Module 9; what's new is the second copy + sync.
**Q5 — True.** A twin is only useful while it stays synchronized.

**Q6 — model answer.** Observation breaks first: a drifted twin no longer reflects the robot's current state, so just *looking* misleads. Experimentation and reasoning build on observation — they take the mirrored state as their starting point — so once the mirror is wrong, they inherit the error. All three depend on faithfulness; observation is the most immediate casualty.
*Grading: credit identifying observation as first, with the dependency argument.*
