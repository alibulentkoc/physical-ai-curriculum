# Lesson 4.7 — Answer Key (instructor only)
**Q1** mirror; size preserved, handedness flipped. **Q2** (2,−3). **Q3 (T/F)** True (det −1). **Q4 (short)** rotation keeps handedness (det +1); reflection reverses it (det −1), so no rotation can produce a mirror image.
**Challenge (§10) rubric:** handedness/determinant argument that reflection ≠ rotation; implication that a left-handed part is not the rotated right-handed part. Credit the orientation-flip reasoning.
