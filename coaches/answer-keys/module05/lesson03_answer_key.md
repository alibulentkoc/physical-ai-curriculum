# Answer Key — Lesson 1.3: Reachability and the Workspace

**Coaches only.** Formative.

1. **Solvable iff** — the target lies in the reachable workspace.
2. **2-link workspace shape** — the annulus |L₁−L₂| ≤ r ≤ L₁+L₂.
3. **Joint limits shrink the workspace** — True.
4. **(0.05, 0) with L1=0.4, L2=0.3 (short).** Not reachable: r = 0.05 < |L₁−L₂| = 0.1, i.e. inside the inner hole.

**Challenge rubric.** Full credit: equal link lengths give |L₁−L₂| = 0, so the inner radius is zero and the annulus becomes a full disk; reaching a point near the base means the arm folds nearly back on itself (θ₂ near 180°). Partial: states "no hole" without the |L₁−L₂| = 0 reason.
