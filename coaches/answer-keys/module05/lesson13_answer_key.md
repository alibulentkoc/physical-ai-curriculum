# Answer Key — Lesson 4.1: When Closed Form Runs Out

**Coaches only.** Formative.

1. **Redundant arm** — has infinitely many solutions (a solution manifold).
2. **What breaks closed form** — general geometry, redundancy, or coupled orientation.
3. **Every arm has an evaluable, differentiable forward map** — True.
4. **Why redundancy is useful (short).** Extra freedom lets the arm reach a target many ways, so it can dodge obstacles or respect joint limits while grasping.

**Challenge rubric.** Full credit: a planar n-link arm reaching a 2D point has n−2 extra DOF, so its solution manifold has dimension n−2; for n ≥ 3 that is a positive-dimensional continuum, which no finite formula can enumerate, so (in general) there is no closed form. Partial: identifies n−2 freedom without tying it to the impossibility of a closed form.
