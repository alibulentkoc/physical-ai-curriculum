Trajectory Studio — Plan → Parameterize → Validate → Execute

The whole Module 7 workflow on the greenhouse arm. Drag the obstacle, the start (green) or goal (black) tool point. Then run the pipeline: Plan routes around the obstacle (RRT), Parameterize times it within the limits, Validate runs the checklist, and Execute plays the validated reference and shows the reference signal (q_d, q̇_d, q̈_d) that hands off to Module 8. Module 7 produces the reference; it does no feedback control.
