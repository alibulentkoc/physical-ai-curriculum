Same path q(s) = 20° + 60°s, two clocks. The geometry (where the joint goes) is identical; the timing s(t) differs. Constant-rate moves uniformly; eased starts and ends gently. Watch q(t).
A path is a curve in space; a trajectory is that path plus a schedule. Re-timing the same path changes velocities and accelerations completely — which is where smoothness and feasibility live.
