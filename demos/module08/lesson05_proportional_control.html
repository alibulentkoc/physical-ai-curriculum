Correction proportional to error: u = Kp·e. Raise Kp and the response gets faster and the gap shrinks — but a proportional controller fighting a load never fully closes it. A residual offset always remains.
At steady state the joint stops, so the only effort left is Kp·e — which must balance the load. That requires a non-zero error. More Kp shrinks it but can't erase it.
