Raise the damping λ. The pseudoinverse inverts each singular value as 1/σ — which explodes as σ→0. Damped least squares uses σ/(σ²+λ²) instead: it rises, peaks, then rolls back to 0, taming the singular direction.
At large σ both agree (1/σ). Near σ = 0 the DLS factor → 0, sacrificing a little accuracy to keep joint rates finite. The peak sits at σ = λ with value 1/(2λ).
