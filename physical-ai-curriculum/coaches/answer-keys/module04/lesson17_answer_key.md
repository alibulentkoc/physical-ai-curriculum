# Answer Key — Lesson 5.1: Why a Convention

**Coaches only.** Formative.

1. **DH convention is** — a shared standard for placing frames, reducing each joint to four parameters.
2. **Why arbitrary frames are a problem** — they give inconsistent, non-comparable transforms.
3. **DH reduces six free choices to four parameters** — True.
4. **Why robots ship with a DH table (short).** So one kinematics codebase can model any arm — only the table changes between robots.

**Challenge rubric.** Full credit: a minimal four-number parameterization is cheaper to store, easier to calibrate (fewer independent quantities), and less prone to inconsistent entries than a full 16-number (6-independent) matrix per joint. Partial: notes fewer numbers without the calibration/consistency reasons.
