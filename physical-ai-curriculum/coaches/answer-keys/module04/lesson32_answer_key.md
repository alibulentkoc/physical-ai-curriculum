# Answer Key — Lesson 8.4 / Module 4: Wrap-Up and the Road to Inverse Kinematics

**Coaches only.** Capstone, formative consolidation.

1. **Forward kinematics is** — a function to evaluate (always defined, one answer).
2. **Inverse kinematics (Module 5) is** — solving T_0^n(θ) = T_desired for θ — possibly 0, 1, or many solutions.
3. **Forward kinematics is the foundation that inverse kinematics inverts** — True.
4. **What Module 4 built (short).** The forward map from joint angles to the gripper pose, T_0^n(θ) = product of DH-generated factors, joined to perception as T(world←arm).
