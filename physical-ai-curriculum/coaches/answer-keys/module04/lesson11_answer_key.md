# Answer Key — Lesson 3.3: A Planar 2-/3-Link Arm

**Coaches only.** Formative.

1. **Gripper position** — sum of link reaches Lk(cosφk, sinφk) at accumulated angles φk.
2. **Orientation** — φn = θ1+θ2+…+θn.
3. **Straightened reach = ΣLk** — True.
4. **3-link (30°,60°,−30°) (short).** Position (0.446, 0.673); orientation 60°.

**Challenge rubric.** Full credit: with one extra link of the same total length the arm gains a DOF, so many configurations reach a given point (redundancy) and the gripper orientation can often be set independently of position. Partial: notes more configurations without the orientation-control point.
