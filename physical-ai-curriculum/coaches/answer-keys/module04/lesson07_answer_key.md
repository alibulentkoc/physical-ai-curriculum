# Answer Key — Lesson 2.3: Where Is the Tip?

**Coaches only.** Formative.

1. **Position in the pose matrix** — the translation column.
2. **Orientation** — the rotation block (upper-left).
3. **Matrix matches trigonometry** — True.
4. **θ=90°, L=0.5 (short).** Position (0, 0.5); orientation 90°.

**Challenge rubric.** Full credit: for the one-joint arm position and orientation are both functions of the single θ, so they cannot vary independently; adding a second joint (or a wrist) provides the extra DOF to decouple them. Partial: states coupling without the fix.
