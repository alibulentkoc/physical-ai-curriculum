# Answer Key — Lesson 5.4: Denavit–Hartenberg Parameters (Unit 5 Recap)

**Coaches only.** Formative consolidation.

1. **A DH table has** — one row of four parameters per joint.
2. **The DH table is** — the robot's kinematic identity.
3. **Frames: z along joint axes, x along common normals** — True.
4. **What Unit 6 does with the table (short).** Turns each row into a link transform and multiplies them to compute forward kinematics T_0^n.
