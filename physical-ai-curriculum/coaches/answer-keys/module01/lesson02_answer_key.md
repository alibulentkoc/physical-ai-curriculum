# Lesson 1.2 — Answer Key & Feedback (instructor only)
**Q1** Number + unit. **Q2 (short)** 250 cm × (1 m/100 cm) = 2.50 m; dimension = length (L). **Q3** length ÷ time (valid; a velocity). **Q4 (T/F)** False — a number needs a unit.
**Challenge (§10) rubric:** output = a·b/c with a=L, b=T, c=L/T → dimension = (L·T)/(L/T) = T². Strong answers do the dimensional algebra and name a plausible quantity; partial credit if they get the method but slip the algebra.
