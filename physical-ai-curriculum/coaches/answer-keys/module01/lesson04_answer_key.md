# Lesson 1.4 — Answer Key & Feedback (instructor only)
**Q1 (short)** true = perfect-instrument value; measured = instrument's reading; abs error = measured − true. **Q2** 2%. **Q3** Systematic. **Q4 (T/F)** False — all sensors err.
**Challenge (§10) rubric:** "grabs low almost every time" → systematic (consistent direction); "varying amounts" → also random. Strong answers identify BOTH components and give matched fixes (calibrate the bias; average/filter the scatter).
