# Lesson 1.3 — Answer Key & Feedback (instructor only)
**Q1** Scalar = magnitude+unit, no direction. **Q2** Direction. **Q3 (T/F)** True (no direction → scalar). **Q4 (short)** Directional quantities need several components (one per axis) → stored as a vector, a different type than a single-number scalar.
**Challenge (§10) rubric:** energy is scalar, force is not (reversing force direction changes the situation; reversing "energy direction" is meaningless). Credit the correct use of the reversal test + a greenhouse example for each; note force needs Unit 2 vectors.
